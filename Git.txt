基本配置
1.打开Git Bash
2.设置用户信息 user.name 后必须加空格
git config --global user.name "Bruce"
git config --global user.email "bruce1163366325@gmail.com"
查看配置信息
git config --global user.name
git config --global user.email

删除用户在删除windows凭证无效后可以使用下面方法删除用户
git config --global --remove user

查看信息
git config --global --list

配置别名 用于简写命令
用户目录下，创建文件 .bashrc
输入：
alias git-log='git log  --pretty=oneline --all --graph --abbrev-commit'
alias git lg="log --color --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit"



配置本地仓库
1.选择本地某个文件夹，项目的最好
2.进入目录中 
3.初始化仓库 git init
4.查看仓库文件状态 git stauts
5.新增文件进入仓库：git add . (点是通配符，所有文件)
6.提交文件进入分支：git commint -m 'add file'
7.修改文件后，需要再执行5.6 步骤

8.版本回退 git reset --hard commitID
9.查看已删除的记录：git reflog


分支：
查看分支：git branch
新增分支dev ：git branch dev (dev是自定义名称)
切换当前分支：git checkout -b dev (-b 表示如果没有该分支，则进行创建)
合并分支：git merge 分支名 (一般情况都是合并到master)
删除分支：git branch -d 分支名(-d 需检查 -D强制删除)

IDEA中设置GIT
1.进入setting 搜索git  进入 git 选项 一般情况下自动识别本地的git 
2.点击test，测试是否自动识别，否则需要手动指定地址
---推入本地仓库---
3.IDEA打开本地项目文件夹，添加.gitignore文件
4.顶部导航栏：VCS -> import into version control -> create git repository...
5.选择第三步中的本地项目
6.顶部导航栏出现git的操作按钮，点击勾选项，全选
7.写入提交备注，提交
8.左下角 versions control 或者 Git  ->log 可查看提交日志

---推入远程仓库---
9.VCS->Git->Push
10.首次push需要添加默认的远端仓库地址
11.点击push推送到远程仓库

---克隆远程仓库到本地---
12.获取到远程仓库的SSH地址
13.VCS->Checkout from Version Control->Git
14.填入SSH地址，修改存放目录即可
